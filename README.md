# The Odin Project / Ruby Course: Chess

This is the capstone project for the Ruby Course in The Odin Project. It is a chess game,
to be played in the terminal. Note that the display may not work properly when using 
Terminal in MacOS. For testing in the MacOS environment, [iTerm2](https://iterm2.com/) 
is recommended. Thanks to Josh Smith for this helpful 
[article](https://dev.to/joshdevhub/terminal-colors-using-ruby-410p) on using colors in 
the terminal.

Work in progress.
